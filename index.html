Structuring content

header is the introductory content at the top of the page.

footer is the ending of the website.

main is element that is the main content of the page. Only used once to tell the browser where the main content.

section is used to wrap around related pieces of content. each section usually has its own heading.

article is an element that is a containted composition in a web page wich is intended to be distributable or reusable.

nav is the element that represents a part of a page whose job it is to provide navigation links to other parts of the document or other documents.

Linking to an element on the same page

linking an element on the same page requires you have a section id on a part of the page. Lets say we are linking to a gallery portion of a website, we would give it the section id="gallery", and then we would begin our linking. We would then use the "a" tool to link it. So we would do "a href="#gallery"> Gallery /a "

Commenting code

commenting is used to just comment your code, and will not display on your browser. They are used by just using pointy brackets and ! and writing your comments in between the brackets.

Landmark roles

Landmark roles are used to help people using screen readers jump to specific sections in websites.

role="banner" is used to define an element as the header. Added to the primary header.

role="navigation" is used to define an element as the primary navigation. Added to the top nav.

role="main" is used to define an element as the main content of the page. Added to the main

role="complimentary" is used to define an element as being content related to the primary content. Added to an aside

role="contentinfo" is used to defininf an element as information about the content like copyright, terms, privacy policy etc. Added to a footer.

Skip links

skip links are put at the top of a page and is a way for users to jump to certain areas of the page. This is used the same way as linking an element on the same page but instead of using a section we would use nav or main to jump to them.

CSS reference

Font properties

font-family is used to change what family of font is used.

font-size is used to change what size the font is used.

font-weight is used to make things differnt weights like bold.

line-height is used to tell the browser how long you want the line to be.

Text decorations

Text-transform is used to transform text by making it all upercase or lowercase for example. (text-transform: uppercase;)

Text-decoration is used to add things like underlines or strikethroughs. (text-decoration: line-through;)

Text-align is used to align text certain ways like right, left or center aligned. (text-align: center;)

Text-indent is used to indent text. (text-indent: 3em;)

Text-shadow is used to add shadow to text. (text-shadow: 2px -4px 6px orange;)

Link states

:link is used to style default link states

:visited is used for styling the link after its been visited.

:hover is used for styling the link when the cursor is ontop of it.

:focus is used for styling the link when the keyboard has tab focused it.

:active is used for styling the link when the mouse button is pressed down on it.

Boxes

Outside content is the padding, pushing the edge of the box away from the content. Gives the content some breathing room

Margin is outside of the box, pushing other boxes away. It seperates the box from other elements.
